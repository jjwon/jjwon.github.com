Fake Miniaturization

This was one of the choices for a final project for my computational photography class. It was particularly interesting to me because I used to adore this effect, and have built my own ghetto tilt-shift lenses in the past.

Doing this in Matlab is slightly different. The user draws a box for the focus region, and this part stays in focus. From here, there are two approaches. The first one blurs the rest of the image in bands corresponding to the distance from the focus region. The second (more intelligent) way blurs the image corresponding to a rudimentary depth map, and blurs things that have a different depth (where depths have been binned). The pieces are then blended together using a Gaussian mask.

Check out the full results on our final webpage!

Project Homeless Connect

Project Homeless Connect is a nonprofit in San Francisco which hosts public events for the 75,000 homeless in San Francisco, where they can get social and medical services. I'm working with four others from Cal Blueprint to build an Android app to make the events that they host more efficient by decreasing registration time, preventing loss of sensitive information, and eliminating post-event data entry.

Check it out on Github!

Portkey

Portkey is a optimization tool I built while I was an intern at LinkedIn during summer 2014. A lot of people on my team would create workflows to move metrics data from sources like Kafka onto a Hadoop server, perform some transformations on it, and then store it into a database to query it and update the company dashboard easily. However, the scheduling software used to write workflows had a poor interface, which made writing workflows take an extremely long time. I built a better user-interface with autocompletion, intelligent parameter passing, and saved workflows, decreasing workflow composition time by more than 90%.
